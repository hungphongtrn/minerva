  <action>
Create `src/tests/services/test_bounded_recovery.py` to validate RunService recovery semantics around gateway execution.

Test requirements (mock at source module level):
1) Recoverable errors (GatewayErrorType.HEALTH_CHECK_FAILED, TIMEOUT, TRANSPORT_ERROR) trigger reprovision via `_recover_routing_target()` and retry, up to 3 total attempts.
