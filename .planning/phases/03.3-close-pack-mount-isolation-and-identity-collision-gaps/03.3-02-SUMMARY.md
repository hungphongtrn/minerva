## Decisions Made

- **Module-level constants for isolation contract** - Rather than scattering path strings throughout the codebase, defined constants in base.py that both providers can import and use
- **Symlink with force flag** - Using `ln -sf` allows re-provisioning scenarios where symlinks may already exist
- **SHA-256 hash for user suffix** - Creates deterministic, collision-resistant 10-character suffix for sandbox refs while keeping them readable
- **Workspace-path verification** - Checking identity files at workspace path (/home/daytona/workspace/) rather than pack path ensures the full chain works: volume mounted → symlinks created → files accessible
