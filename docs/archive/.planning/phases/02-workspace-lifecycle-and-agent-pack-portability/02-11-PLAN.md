<notes>
Key Daytona Async SDK patterns to use:
- Import: `from daytona import AsyncDaytona, DaytonaConfig`
- Initialization: `async with AsyncDaytona() as daytona:` or `daytona = AsyncDaytona(config)`
- Create sandbox: `sandbox = await daytona.create(params=None, timeout=60)`
- Get sandbox: `sandbox = await daytona.get(sandbox_id)`
- Start/Stop: `await daytona.start(sandbox)`, `await daytona.stop(sandbox)`
- Delete: `await daytona.delete(sandbox)`
- Sandbox object has properties: `id`, `state`, `status`, `target`, etc.
</notes>
