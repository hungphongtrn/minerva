## Decisions Made

- Auto-stop interval defaults to 0 (disabled) for production runtime continuity
- Identity verification hard gate requires sandbox to be in running state
- Gateway resolution follows priority: explicit metadata -> preview URL derivation -> constructed URL
- 3 reprovision attempts with exponential backoff (1s, 2s, 4s capped at 5s)
- Hydration failures are non-blocking - they mark degraded status without affecting routing success
- Bridge token grace period of 30 seconds prevents in-flight request failures during cutover
