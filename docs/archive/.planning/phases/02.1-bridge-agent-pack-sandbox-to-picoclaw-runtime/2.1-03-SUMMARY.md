## Decisions Made

### Session Scoping
- **Authenticated runs:** Session key = `minerva:{workspace_id}:{agent_pack_id}:{run_id}`
- **Guest runs:** Session key = `minerva:guest:{run_id}` (ephemeral, per-request unique)

### Bridge Error Mapping
| Bridge Error Type | HTTP Status | Remediation |
|-------------------|-------------|-------------|
| HEALTH_CHECK_FAILED | 503 | Sandbox unhealthy, retry or reprovision |
| AUTH_FAILED | 503 | Check bridge token configuration |
| TIMEOUT | 504 | Increase timeout or check sandbox performance |
| TRANSPORT_ERROR | 503 | Check network connectivity |
| UPSTREAM_ERROR | 502 | Check Picoclaw gateway logs |
| MALFORMED_RESPONSE | 502 | Contact support |
