# Development Process

Markdown-first process guidance for turning ideas into shippable work with progressive disclosure and beans.

## Process Docs

- `docs/process/markdown-beans-workflow.md`: end-to-end workflow from idea to MVP, phases, canonical docs, and bean-sized tasks.
- `docs/process/bean-memory-policy.md`: when to create or reuse beans, and how beans act as operational memory for stateless agent sessions.
- `docs/process/bean-template.md`: standard bean body template for resumable execution and handoff.
